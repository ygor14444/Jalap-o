Este projeto consiste em um website desenvolvido com HTML e CSS, com o objetivo de apresentar o destino turístico Jalapão, destacando suas principais belezas naturais e informações relevantes.

Funcionalidades
Estrutura de página com layout organizado
Menu de navegação simples e intuitivo
Galeria de imagens com pontos turísticos
Seção informativa sobre o Jalapão
Design limpo e responsivo básico
Conteúdo

O site exibe imagens de locais famosos da região, como:
Dunas do Jalapão
Fervedouros
Cachoeiras

Além disso, apresenta um resumo sobre a história e a importância econômica do turismo na região.
Tecnologias utilizadas
HTML5
CSS3
Objetivo do projeto

Este projeto foi desenvolvido como prática de desenvolvimento web, com foco em:

Estruturação de páginas com HTML
Estilização com CSS
Organização de layout
Criação de interfaces simples
